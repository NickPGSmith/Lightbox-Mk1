## Code

The code was written in C++ with the Arduiono framework, developed and compiled with the PlatformIO plugin for VSCode.

* main.cpp : Main code.
* colours.h / colours.cpp : Define pallette and colour functions. 
* pins.h : Define Arduino pin numberings.
* storage.h / storage.cpp : Functions to get/set settings to the EEPROM.
